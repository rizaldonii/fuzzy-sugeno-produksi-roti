# Fuzzy Sugeno Project

Simple Streamlit UI + placeholder fuzzy engine.
Project untuk demo Sugeno fuzzy ranking.

## Run locally

1. Buat virtualenv:
   - `python -m venv .venv`
   - aktifkan `.venv\Scripts\Activate.ps1` (Windows PowerShell) atau `source .venv/bin/activate` (mac/linux)
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Jalankan:
   - `streamlit run app.py`
